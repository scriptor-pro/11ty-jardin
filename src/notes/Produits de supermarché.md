# Produits en vente dans un supermarché

| Produits favorables (effet exploitable) | Produits défavorables (effet négatif) |
|-----------------------------------------|----------------------------------------|
| **Bicarbonate de soude** → augmente le pH, booste le contraste et le grain | **Vinaigre (excès)** → pH trop bas, développement écrasé et plat |
| **Soude en cristaux (carbonate de sodium)** → effet plus marqué, accélère le développement | **Jus de citron / agrumes** → trop acide, taches et contraste réduit |
| **Sucre blanc / brun** → adoucit, effet compensateur (lumières lissées) | **Huiles alimentaires (olive, tournesol, etc.)** → film gras, zones mal développées |
| **Miel** → effet sucré similaire au sucre, plus imprévisible | **Lait, crème** → graisses et protéines qui voilent l’émulsion |
| **Eaux minérales très minéralisées** (Hépar, Contrex, Vichy Célestins) → micro-contraste, grain accentué | **Eau de Javel, nettoyants chlorés** → détruisent instantanément l’image |
| **Sel (très faible dose)** → freine légèrement, effet compensateur | **Sel (forte dose)** → inhibition excessive, négatifs plats ou effacés |
| **Boissons sucrées légères (cola dilué, sirop)** → possibles teintes, adoucissement | **Boissons très sucrées / collantes** (sodas concentrés, confitures, sirops épais) → dépôts, collages, taches |
| **Colorants doux** (ex. bleu de méthylène pharmacie) → possible coloration de l’émulsion | **Lessives, produits ménagers** → trop alcalins, enzymes et tensio-actifs détruisent la gélatine |
